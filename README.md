# KidsGuard – Official Landing Page

This repo hosts the official landing page for the [KidsGuard Chrome Extension](https://chrome.google.com/webstore), served via **GitHub Pages**.

🌐 **Live site**: https://mcsrichard.github.io/kidsguard/

## Structure

```
index.html   — Main landing page
style.css    — Stylesheet
assets/      — Icons and images
```

## Deployment

Any push to `main` automatically deploys via GitHub Pages.  
Enable in: **Settings → Pages → Deploy from branch `main` / (root)**

## Extension

👉 Install on [Chrome Web Store](https://chrome.google.com/webstore)  
📦 Extension source: [mcsRichard/KidsGuardExtension](https://github.com/mcsRichard/KidsGuardExtension)
